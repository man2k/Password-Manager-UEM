## Install Neutralino CLI:

```bash
npm install -g @neutralinojs/neu
```

## NeutralinoJS dev commands:

```bash
neu update
```

```bash
cd extensions/nodeServer && npm install && cd ../../frontend/ && npm install && cd ../
```

```bash
neu run
```

### OR

```bash
neu run -- --window-enable-inspector
```

## NeutralinoJS Build Commands:

```bash
neu build
```
